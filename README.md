Court Booking System

Project Overview

Court Booking System is a web-based application developed to simplify the process of booking sports courts. The system allows users to view available courts, select a court type, choose a booking date and time, make a booking, and view their booking history. It also includes an admin section for managing courts, monitoring bookings, and updating payment status.

This project was developed as part of a Software Engineering and Database project. The main objective is to demonstrate a complete court booking workflow with a user-friendly interface, booking management, payment tracking, and database integration. The live GitHub Pages version is mainly used to demonstrate the website interface, while the full backend version requires Flask and Oracle Database to run locally.

Features

* View available sports courts
* Filter courts by court type
* Book a court based on date, time, and duration
* Select payment method
* View booking history
* Admin dashboard for booking and court management
* Payment status tracking
* Backend support using Flask and Oracle Database

Project Purpose

The purpose of this project is to provide a simple and organized platform for students or users to book sports courts more efficiently. Instead of using manual booking methods, users can check court details, submit booking information, and manage their booking records through the system.

Demo Note

The deployed GitHub Pages version is intended for interface demonstration only. Some features use mock data or localStorage because the Flask backend and Oracle Database need to be run locally for full system functionality.
